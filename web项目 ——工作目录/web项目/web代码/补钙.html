补钙

补钙吃什么食物，补钙食谱助您补钙的治疗

人体中99%的钙质存在骨骼牙齿中，支持人体的运动和咀嚼能力。体内缺钙，会影响身体成长，导致神经性偏头痛、烦躁不安、失眠，还会出现肌肉痉挛的症状。经常盐酸背痛、血糖低或者常饮用碳酸饮料的人群需要补钙。补钙的最佳方式多在日常饮食中选择天然的食物，如牛奶和豆制品，其钙含量绝对能够满足一个人的需要。
